SIGNUM — HOW TO READ THIS

A 60-second guide to the Job Market Signal Terminal

What is Signum?

Signum pulls live job postings from real sources (Remotive, Arbeitnow, Hacker News), extracts the skills mentioned in each, and watches which pairs of skills are rising together. It's a signal terminal for spotting what's becoming hot in the job market — not a job board.

[1] SIGNALS — what's breaking out

The main view ranks skill pairs by a Breakout score. Think of it as "which two skills are suddenly showing up together more than usual."

Each row shows:

• Breakout — trend strength. 3.22 = a strong surge. Formula: velocity × novelty × log(count+1).
• Vel. (+N more / 30d) — how many extra postings mentioned this pair in the last 30 days vs the 30 before. +4 more/30d = 4 more than before.
• Nov. — how new the pair is. A pair first seen recently scores higher (fresh signal, not an old habit).
• Trend — a mini sparkline of the pair's recent posting count.

Confluence × Jira · 3.22 breakout · +4 more / 30d
= this tool-pair appeared in 4 extra postings last month and is among the hottest combinations right now.

[2] SEARCH — prove it's real

Click a skill chip (or type one) to find live postings that mention it. Every result has a Verify ↗ link straight to the original job on the source site — so you can confirm the data is real, not made up. If a search misses, Signum suggests a real skill you might have meant.

[3] GRAPH — how skills connect

A network of which skills appear in the same postings. The amber node is the skill you searched; green nodes are skills that co-occur with it. Thicker lines = stronger connection. Click OPEN FULLSCREEN for an animated, draggable view.

[4] SYSTEM — is it healthy?

Shows live status of the API, TimescaleDB, and the Redis bitmap index, plus the list of API endpoints powering everything.

The ticker up top

The scrolling bar shows the top breakout pairs right now — a real-time headline of what's trending.
